HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • H1 to H6 are different font sizes for the webpage headers

CSS

  • A margin indicates how much space we want around the ouside of an element
  • A padding indicates how much space we want around the content inside an element
  • HTML hex code websites can be used to find the right color
  • Link CSS with HTML in header after meta before title
  • The CSS Rule is "div {property:property value;}" The property and property value make up the Declaration continue with the pattern for each attribute you want to give until all the declarations are made.
  • The wildcard * is used to apply the attribute to the entire page use before the brackets
  • Use a comma to add like declarations to different parts of the website

Git

  • git status: checks what branch we are currently on
  • git checkout: -b branch-name: creates a new branch and switches to it
  • code .: opens VS code
  • cd: changes directories going forward
  • cd ..: changes directories going backward
  • git checkout main: moves you to the main branch
  • To send changes to github make sure in command you are in the correct directory. Then type git add -A > Enter then git commit -m "enter note here" > Enter. Then type git pull origin main > Enter. Then git push origin name of branch.

Javascript

  • JavaScript reads from the top to the bottom, order matters
  • A variable is a named container that allows us to store date in our code.
  • Control flow is the order in which a computer executes code in a script
  • Capitalization matters in JavaScript
  • Ending with ; is necessary
